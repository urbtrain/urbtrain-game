# URBTRAIN Game

Protótipo de jogo 3D endless runner da URBTRAIN, otimizado para celulares e publicado com GitHub Pages.

## Tecnologias

- Babylon.js
- TypeScript
- Vite
- GitHub Pages

## Controles

- Arraste para esquerda/direita para trocar de faixa
- Arraste para cima para pular
- Teclado: setas esquerda, direita e cima

## Executar localmente

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
